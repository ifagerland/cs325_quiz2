# cs325_quiz2

## To run program:

    python3 ./calculator.py

## To add:
    Select option 1 when prompted
    Enter first number
    Enter second number

## To subtract:
    Select option 2 when prompted
    Enter first number
    Enter second number
    First number will be subtracted by the second number

## To multiply
    Select option 3 when prompted
    Enter first number
    Enter second number

## To divide
    Select option 4 when prompted
    Enter first number
    Enter second number
    First number will be divided by the second number
    The second number cannot be 0

## To exit
    Select option 5 when prompted
